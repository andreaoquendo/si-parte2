# Tarefa 2: Regressão e Classificação

# Arquivos

- redesNeuraisAlg.py : parametrização do algoritmo por redes neurais
- RNGravidadeClassificacao.py : verificação das métricas por redes neurais utilizando um conjunto 'sinais_vitais_teste.txt' com *cabeçalho*
- RNGravidadeClassificacao.py : utilizado para gerar a análise de gravidade e risco por redes neurais em formato csv utilizando um conjunto de dados'sinais_vitais_teste.txt' com *cabeçalho*.

- CARTAlg.py : parametrização do algoritmo por CART
- CARTGravidadeClassificacao.py : verificação das métricas por CART utilizando um conjunto 'sinais_vitais_teste.txt' com *cabeçalho*
- CARTGravidadeClassificacao.py : utilizado para gerar a análise de gravidade e risco por CART em formato csv utilizando um conjunto de dados'sinais_vitais_teste.txt' com *cabeçalho*.

## Instruções de execução

Para testar novos dados, usar por nome de arquivo 'sinais_vitais_teste.txt' com *cabeçalho*, e executar o arquivo em python "RNGravidadeClassificacao.py" para a análise com CART, que emitirá o document dados_neurais.csv.

Para testar novos dados, usar por nome de arquivo 'sinais_vitais_teste.txt' com *cabeçalho*, e executar o arquivo em python "CARTGravidadeClassificacao.py" para a análise com CART, que emitirá o document dados_neurais.csv.
